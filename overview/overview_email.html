Home Page Features

Background Customization

Use the form on the home page to change the home page background to your liking. Change pattern, primary and secondary colors, and shape size.

Personalized Website Settings

Go to the website settings page to select a location and time zone for use in various website features. This website uses the Bing Maps API as the mapping control and time zone lookup service.

Solar Times and Weather

Select the "track time and weather" option in the home page to grab data from the National Weather Service API and obtain sunrise and sunset times through NOAA Solar Calculations for use in the background.

Featured Projects

SimCity 4 Buildings

A SimCity 4 buildings list database project. Add, view, edit, and delete SimCity 4 buildings using the website forms.

Posts

Add small blog posts to the website using forms and have people comment on them.

Tactics Boards

Create sports boards for basketball, soccer, or American football. Place players and the ball and create paths in order to save set plays so that they can be modified and practiced.
